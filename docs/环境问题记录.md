# 环境问题记录

本次 ConsisID 官方 VBench v1 评测不是 16 个维度全部成功完成。主要问题集中在依赖、模型下载和 CUDA/PyTorch/Detectron2 兼容性。

## 已观察到的问题类型

- `pkg_resources` 缺失或 setuptools 兼容问题。
- Hugging Face 下载超时。
- Azure Blob 下载出现 409。
- GitHub 下载或安装触发 429 限流。
- detectron2 安装/编译失败，和 CUDA/PyTorch 版本匹配有关。
- 部分 VBench 维度需要额外模型或检测器，环境未完全就绪时会失败。
